# myNotes CLI 📝

A robust, lightweight terminal-based note management system written in Pure Bash. Designed for developers and power users who live in the CLI.

## ✨ Features
- **Full CRUD Support:** Create, Read, Update, and Delete notes directly from your terminal.
- **Fast Search:** Built-in keyword search across all your notes using optimized grep logic.
- **Cloud Sync Ready:** Works seamlessly with Dropbox, OneDrive, or any local sync folder.
- **Multilingual Support:** Interface available in both English and Greek.
- **Professional Installer:** Includes an install.sh script for easy global deployment.

## 🚀 Installation
1. Clone the repository:
   git clone https://github.com/YOUR_USERNAME/myNotes-CLI.git
   cd myNotes-CLI

2. Run the installer:
   chmod +x install.sh
   ./install.sh

3. Follow the prompts to set your preferred notes directory and text editor (e.g., nano, vim).

## 🛠 Usage
Simply type `myNotes` from any terminal location.
* Press `w` to write a new note.
* Press `r` to read a note.
* Press `s` to search your database.
* Press `e` to edit an existing note.
* Press `d` to delete a note.
